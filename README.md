# medlabweb

## Usage

- Run this commmand to bring up the docker image
```
docker compose up
```

- Access the Web server with:
```
http://localhost
```

- phpmyadmin can be access with:
```
http://localhost:8080
```

## Directory 

- www/
  - contain the front-end source code
- mysql/
  - cotain the mysql data with table that front-end will use
